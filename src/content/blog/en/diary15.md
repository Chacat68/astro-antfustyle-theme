In the internet era, browsers have become an indispensable part of daily life. Browser extensions are software that adds functionality to browsers, providing users with a better online experience. In this article, I will introduce some commonly used browser extensions.

---

## uBlock Origin

uBlock Origin is a lightweight ad-blocking extension that automatically blocks ads, trackers, and malware on webpages. It performs excellently and consumes less memory compared to similar extensions.

## 1Password

1Password is a password management extension that helps users generate strong passwords, save passwords, and auto-fill forms. It is a stable password software extension that supports all platforms.

## Immersive Translate

Immersive Translate is a language learning extension that provides users with a more natural and fluent translation experience. It supports translation in multiple languages and allows translation while browsing webpages, eliminating the need for manual copy-pasting. This extension helps users better read and learn foreign language materials.

- Subtitle translation for YouTube videos
- Webpage translation
- Supports translation services like DeepL and OpenAI

## Violentmonkey

Violentmonkey is a script management extension that allows users to implement custom features by writing scripts, such as auto-filling forms or skipping ads. This extension requires some programming skills, but once mastered, it can bring great convenience to users.

## FigmaCN

FigmaCN is a design extension that translates the Figma interface into Chinese, making it more convenient for Chinese users to design with Figma. This extension improves design efficiency and reduces the impact of language barriers.

## Minimal Theme for Twitter

Recently, I've been spending more time scrolling through Twitter. This is a modern-style extension that simplifies the Twitter interface.

## Notion Web Clipper

An official tool for temporarily saving articles you want to read.
