## Previous Years' Recommendations:

### [**1Password - Password Manager**](https://chromewebstore.google.com/detail/1password-%E5%AF%86%E7%A0%81%E7%AE%A1%E7%90%86%E5%99%A8/aeblfdkhhhdcdjpifhhbdiojplfjncoa)

1Password is a popular password manager that helps users securely store and manage their passwords, credit card information, online accounts, and other sensitive data.

### [**Eagle for Chrome**](https://chromewebstore.google.com/detail/lieogkinebikhdchceieedcigeafdkid)

This extension allows you to easily save images and screenshots to the Eagle App. When browsing websites, you can instantly save interesting pictures to Eagle and quickly organize them into categories.

### **~~FigmaCN~~**

~~Chinese Figma plugin, manually translated and verified by designers.~~

### [FigmaEX](https://chromewebstore.google.com/detail/figmaex/dimbmiiiefochhhipipkphcfjckchhjd)

FigmaEX enhances your Figma experience.

### [**uBlock Origin**](https://chromewebstore.google.com/detail/ublock-origin/cjpalhdlnbpafiamejdnhcphjbkeiagm)

An efficient request filter: uses very little memory and CPU. Compared to other common filtering tools, it can load and execute thousands of filtering rules.

### **~~Violentmonkey~~**

~~An open-source userscript manager supporting many browsers.~~

### [Tampermonkey V3](https://chromewebstore.google.com/detail/%E7%AF%A1%E6%94%B9%E7%8C%B4/dhdgffkkebhmkfjojejmpbldmpobfkfo)

Freely change the web with userscripts to enhance your browsing experience, using Tampermonkey! (Updated to V3 protocol, compatible with the latest Chrome.)

### [**Immersive Translate**](https://chromewebstore.google.com/detail/%E6%B2%89%E6%B5%B8%E5%BC%8F%E7%BF%BB%E8%AF%91-%E7%BD%91%E9%A1%B5%E7%BF%BB%E8%AF%91%E6%8F%92%E4%BB%B6-pdf%E7%BF%BB%E8%AF%91-%E5%85%8D%E8%B4%B9/bpoadfkcbjbfhfodiogcnhhhpibjhbnh)

A free bilingual (original/translated) webpage translation plugin. Supports PDF translation (preserves layout), video subtitle translation (YouTube, Netflix), EPUB e-book translation; supports AI translation models (OpenAI (ChatGPT), DeepL, Gemini(Bard), etc.).

## This Year's Recommendations:

### [**Kimi Copilot - Web Summary Assistant**](https://chromewebstore.google.com/detail/kimi-copilot-%E7%BD%91%E9%A1%B5%E6%80%BB%E7%BB%93%E5%8A%A9%E6%89%8B/icmdpfpmbfijfllafmfogmdabhijlehn)

### [**Elmo - Your AI Web Copilot**](https://chromewebstore.google.com/detail/elmo-chat-your-ai-web-cop/ipnlcfhfdicbfbchfoihipknbaeenenm)

Elmo is your AI Chrome extension for creating summaries, insights, and expanding knowledge.

### **~~Clearly Reader - Reading Assistant~~**

~~A browser extension offering reading mode, copy, translate, text-to-speech, PDF, and Markdown export features.~~

### [Reader View](https://chromewebstore.google.com/detail/reader-view/ecabifbgmdmgdllomnfinbmaellmclnh)

Strips away buttons, background images, and other clutter, changes text size, contrast, and layout of pages to improve readability, using less memory.

### [**BewlyBewly Bilibili Assistant**](https://chromewebstore.google.com/detail/bewlybewly/bbbiejemhfihiooipfcjmjmbfdmobobp)

BewlyBewly is a browser extension for Bilibili designed to enhance the user experience by redesigning the Bilibili user interface. Inspired by YouTube, Vision OS, and iOS, it creates a more visually appealing and user-friendly interface.

### **~~Yellow Tweet Killer~~**

### [Twitter Bird Shield](https://chromewebstore.google.com/detail/igapbfjkbkmjcmgjmgfcegamhkfppdmg)

This plugin can block malicious users (e.g., pornographic, scam accounts) and optimize the interface. It also automatically expands secondary lists, provides a search assistant, video download, and user notes!
