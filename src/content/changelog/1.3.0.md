### 🚀 新功能

- **AI 画廊**：新增 `/gallery` 画廊页面，PhotoView 重构为通用画廊组件，支持展示 AI 生成图片。
- **背景动画扩展**：新增 Constellation（星座）、Firefly（萤火虫）、Wave（波浪）等背景动画效果；首页最终固定为 wave 动画，保证体验一致。
- **项目展示更新**：新增「江湖」武侠卡牌游戏原型、「卧龙吟2」「乱轰三国志」等游戏开发项目。
- **内容校验**：新增翻译版 Markdown 文件的 frontmatter 验证脚本，统一文章标题与日期格式。
- **社交链接**：新增 NeoDB 社交链接入口。

### 🐞 修复

- **SEO 全面修复**：补齐 meta description、canonical 强制 HTTPS、结构化数据、标题长度与 robots 标签。
- **域名迁移**：生产域名更新为 foo-z.com，修正全站站内链接，并优化统计脚本与搜索安全策略。
- **移动端导航**：修复移动端导航 Backdrop 透明度问题，改用不透明背景配合模糊效果；优化 NavSwitch 面板开关的可访问性。
- **CI 工作流**：启用 fork 仓库 CI，防止工作流在其他仓库中意外触发。

### 💅 重构

- **i18n 补齐**：全站排查并补齐国际化缺口，重构重复代码。
- **品牌统一**：将 CHACAT/Chacat 显示名统一替换为 Charliefoo。
- **样式回归**：网站配色与 markdown 配色回归默认黑白灰，移除 Matrix 代码雨等冗余背景效果。

### 🏡 杂项

- **内容更新**：新增多篇中英文博客——《广州生活的慢与厚》《卡片式思考》、广东粤西"年例"新年习俗、AI 游戏开发、AGA 音乐等；重写 2024 年度反思（反脆弱主题）。
- **文档完善**：补充部署发布流程文档，约定先查 GitHub 再查 Cloudflare 的排查顺序。
- **首页调整**：首页文章数量显示调整为 10 条。
