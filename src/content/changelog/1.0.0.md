### 🚀 新功能（目录）

- **站点结构与页面**：新增导航体系与多页面（Blog / Projects / Changelog / Feeds / Streams 等），并提供可复用的列表、分组、卡片等视图组件。
- **主题与交互**：支持浅色/深色模式（含跟随系统与可访问性优化），并接入 View Transitions；新增返回顶部、返回链接、加载进度等交互能力。
- **内容能力增强**：基于 Content Collections + Zod Schema 规范 frontmatter；支持阅读时间统计、KaTeX 数学公式、MDX 写作。
- **Markdown/渲染链路**：完善 rehype/remark 能力（标题锚点、外链处理、callouts、指令扩展、视频/徽章/样式化链接等）。
- **媒体与分享/SEO**：图片容器与缩放预览；支持自动/自定义 OG Image；完善分享平台支持与 SEO（OG 类型、JSON-LD、sitemap/robots）。
- **视觉与工程化**：新增多种背景效果（plum/dot/particle/rose 等）与相关配置；集成 expressive-code；完善 PWA、RSS、CSS/UnoCSS、Lint/CI 等工程配置。

### 🐞 修复（摘要）

- 构建后路由/导航路径解析问题。
- 搜索结果可滚动与滚动条样式问题。
- 多处可访问性与响应式布局问题（包含 projects 视图与 footer）。
- 背景自定义元素初始化时机与可禁用配置。
- 指令相关正则匹配与格式化忽略规则。

### 💅 重构（摘要）

- 配置与自定义项逐步从 JSON Schema 迁移到 TS，并统一为 UI 配置体系。
- 多个页面从 `.astro` 重构为 `.mdx`（动态路由页面保留 `.astro`），并抽象通用渲染/视图组件。
- 背景系统重构为自定义元素形态，便于按页面开关与复用。

### 🏡 杂项（摘要）

- 初始化与基础设施：依赖管理（pnpm）、格式化/校验（prettier/eslint/lint-staged）、构建与 CI/模板等。
- 文档与资源：更新 README/文档、补充 OG 图片与样式整理。
